WEB DEVELOPMENT SKILLS

Learn HTML

HTML stands for Hyper Text Markup Language. HTML is the content of a web page. Examples of HTML includes headings, paragraphs, images, links, lists, etc.

Check out the HTML Tutorial on w3school.com.

Learn CSS

CSS stands for Cascading Sheets. CSS is the style of a web page. Examples of CSS includes font style and color, background color, borders, dimensions, layout, etc.

Check out the CSS Tutorial on w3school.com.

JavaScript

is the programming Language of the web. JavaScript helps to make pages interative and dynamic . Some common JavaScript tasks are:

  • add new HTML to the page,
  • change the existing content / style,
  • react to user events (mouse clicks, key presses, etc.)

Check out the JavaScript Tutorial on w3school.com.
